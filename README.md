## ğŸš€ Getting Started

### Prerequisites

No special prerequisites needed! Just a modern web browser.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Karmoshaar/website-Project.git
```

2. Navigate to the project directory:
```bash
cd website-Project
```

3. Open `index.html` in your web browser:
   - Simply double-click the file, or
   - Use a local server (recommended):
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

4. Open your browser and navigate to `http://localhost:8000`

## ğŸ¯ Customization

To customize this portfolio for your own use:

1. **Update Personal Information**:
   - Edit the name in `main.js` (line 1)
   - Update the About Me section in `index.html` (line 42-45)
   - Modify contact information in `index.html` (lines 77-84)

   - Replace images in `assets/imgs/` with your project screenshots
   - Update project links in `index.html` (lines 57-64)

3. **Update Social Media Links**:
   - Modify the social media URLs in `index.html` (lines 87-98)

4. **Change Colors**:
   - Update color values in `styles.css` to match your brand
